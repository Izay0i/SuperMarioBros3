## Controls

|Key          |Action             |
|-------------|-------------------|
|<kbd>A</kbd> |Move left          |
|<kbd>D</kbd> |Move right         |
|<kbd>S</kbd> |Crouch/Duck        |
|<kbd>J</kbd> |Attack/Hold shell  |
|<kbd>K</kbd> |Jump               |
